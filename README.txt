Веб страница поиска вакансий на портале hh.ru
В форме поиска вводится искомый параметр, далее при помощи парсера будут получены требуемые данные.
На странице результатов будет список найденных вакансий, с ключевыми полями по каждой из них, это: город, название фирмы,
должность, ЗП, ссылка на саму вакансию.
